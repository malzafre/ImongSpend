# ImongSpend Chrome Extension

This folder is the unpacked extension build.

## Install (Developer Mode)

1. Download the release zip from GitHub Releases.
2. Extract the zip to a folder on your computer.
3. Open `chrome://extensions` in Google Chrome.
4. Turn on **Developer mode** (top-right).
5. Click **Load unpacked**.
6. Select the extracted folder (the folder that contains `manifest.json`).

## Update to a New Version

1. Download and extract the new release zip.
2. Open `chrome://extensions`.
3. Find **ImongSpend** and click **Reload**, or remove and load the new extracted folder.

## Notes

- This is an MVP build focused on Shopee-first spend visibility.
- If Chrome shows "Manifest file is missing or unreadable", you selected the wrong folder.
